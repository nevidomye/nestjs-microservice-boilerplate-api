# Documentation

### Building and Running the documentaion

Documentation:
[documentation](https://typespec.io/)

---

- install dependencies
  ```
  $ pnpm run doc:install
  ```
- running infra
  ```
  $ pnpm run infra
  ```
- starting doc
  ```
  $ pnpm start
  ```
